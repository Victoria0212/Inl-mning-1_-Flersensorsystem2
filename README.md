# Flersensorsystem

## Uppgiftsbeskrivning

Kursuppgiften går ut på att bygga ett program med C++ som simulerar flera sensorer, samlar in och lagrar mätvärden, visar enkel statistik per sensor och låter användaren sköta allt via ett menygränssnitt med läs/skriv till fil.

## Funktionalitet

* Skapa simulerade mätvärden i flera sensorer
* Visa upp alla mätvärden som har skapats
* Visa upp statistik över mätvärderna
* Spara ner mätvärderna till en CSV-fil
* Läsa in mätvärden från en CSV-fil
